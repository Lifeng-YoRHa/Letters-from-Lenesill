# 第一章地图设计（方案C：回环迷宫）

## 设计意图

- **高层可以跳回低层**：存在从 Layer 6 回 Layer 4、从 Layer 5 回 Layer 3、从 Layer 3 回 Layer 1 的"逆向通道"。
- **局部子网密集**：Layer 3~6 形成两个互相交织的回环子网，玩家可能在中间段反复绕圈。
- **安全屋作为回环枢纽**：安全屋不仅是恢复点，还是跨层跳跃的锚点。

## 结构规则

- 7 层中间结构（Layer 1 ~ Layer 7），加起点和 Boss，共 36 节点。
- 层号是视觉标签，连接完全自由，**允许回环**（高层连低层）。
- 连线全部双向。

## 回环结构说明

本方案存在 4 条核心回环通道：

1. **L3_2 ↔ L1_4**：从 Layer 3 的安全屋可以直接跳回 Layer 1 的废墟。意味着玩家在安全屋恢复后，可以回头去探索早期漏掉的资源点。
2. **L5_2 ↔ L3_2**：从 Layer 5 的黑市跳回 Layer 3 的安全屋。形成"黑市-安全屋"循环，玩家可以在这两个节点之间反复横跳（如果体力允许）。
3. **L6_2 ↔ L4_2**：从 Layer 6 跳回 Layer 4。L6_2 是死路节点（不连 Boss），但它连回了 Layer 4 的废墟区。
4. **L6_4 ↔ L4_4**：另一条 Layer 6 → Layer 4 的回环。L6_4 也不连 Boss。

**回环造成的迷宫效应**：
- 西北子网：L1_4 ↔ L2_4 ↔ L3_2 ↔ L4_2 ↔ L5_2 ↔ L6_2，带回环。
- 东南子网：L3_4 ↔ L4_4 ↔ L5_4 ↔ L6_4，带回环。

玩家在 Layer 4~6 之间移动时，可能因为回环的存在而反复经过同一区域，产生"绕圈子"的体感。

## 成品路径示例

### 路径A：标准推进线（不走回环）

```
START
  -> L1_2 普战
  -> L2_1 普战
  -> L3_1 艰战
  -> L4_1 普战
  -> L5_1 突发
  -> L6_1 艰战
  -> L7_1 艰战
  -> BOSS
```

**特点**：全程直线，不碰任何回环和死路。代价是错过安全屋、黑市和所有资源点。是一条"裸奔"路线。

---

### 路径B：回环发育线（反复利用安全屋+黑市）

```
START
  -> L1_3 突发
  -> L2_3 黑市
  -> L3_3 委托（接取）
  -> L4_3 突发
  -> L4_2 废墟
  -> L3_2 安全屋（**回环！** 恢复体力）
  -> L1_4 废墟（**回环到Layer 1！** 搜刮早期资源）
  -> L3_2 安全屋（再次恢复）
  -> L5_2 黑市（**回环到Layer 5！** 补货）
  -> L4_2 废墟（退回 Layer 4）
  -> L5_1 突发
  -> L6_1 艰战
  -> L7_1 艰战
  -> BOSS
```

**特点**：利用 L3_2 作为跨层枢纽，在 Layer 1、3、5 之间反复跳跃。3 次恢复 + 2 次黑市访问，资源拉满但体力消耗惊人。没有手电筒指引的话，新手会在子网里彻底迷路。

---

### 路径C：迷宫陷阱线（踩进死路后靠回环逃生）

```
START
  -> L1_5 普战
  -> L2_5 突发
  -> L3_5 突发
  -> L4_5 道路
  -> L5_5 道路
  -> L6_5 道路
  -> L7_2 突发（**死路！** 不连 Boss）
  -> L6_2 突发（退回）
  -> L4_2 废墟（**回环到Layer 4！** 绕过死路）
  -> L5_2 黑市
  -> L6_3 普战
  -> L7_3 道路
  -> BOSS
```

**特点**：误入了 L7_2 死路后，没有原路退回 Layer 6 再绕，而是通过 L6_2→L4_2 的回环直接跳回 Layer 4，从西北子网重新找路。展示了回环的"逃生"价值——死路不是终点，可以通过跨层跳跃绕出来。

## 设计说明

1. **回环=选择权，也是迷路源**：西北子网的 L3_2 是一个超级枢纽（连 Layer 1、2、3、4、5），玩家可以在这里决定回头发育还是继续前进。但没有地图记忆的话，很容易在同一子网里打转。
2. **死路+回环的组合拳**：L7_2 和 L7_4 是死路，但通往它们的 L6_2 和 L6_4 分别带回了 Layer 4 的通道。这意味着"踩进死路"不等于"浪费步数"——如果能利用回环，死路反而成了跨层捷径。
3. **安全屋的战略价值**：L3_2 因为回环的存在，成了全图最重要的战略节点。它不仅恢复体力，还打开了通往 Layer 1 和 Layer 5 的传送门。
4. **与手电筒的联动**：手电筒探查 2 个节点类型，在这个迷宫里价值极高——不仅能避开死路，还能发现回环通道的存在。
