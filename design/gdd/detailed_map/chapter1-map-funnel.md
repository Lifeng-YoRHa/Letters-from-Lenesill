# 第一章地图设计（方案B：漏斗绞杀）

## 设计意图

- **入口宽敞，出口极窄**：前半段分支众多，中后段急剧收缩，只有极少数节点能真正推进到下一章。
- **绞杀点**：Layer 4 是一个"缓冲区迷宫"，玩家可以在里面探索，但只有唯一一个节点能离开。
- **终盘赌博**：Layer 6 到 Layer 7 的 4 个出口中，只有 2 个能见到 Boss，另外 2 个是死路。

## 结构规则

- 7 层中间结构（Layer 1 ~ Layer 7），加起点和 Boss，共 36 节点。
- 层号是视觉标签，连接完全自由。
- 连线全部双向，可自由回退。

## 成品路径示例

### 路径A：南线捷径（相对安全，但暗藏岔路）

```
START
  -> L1_5 道路
  -> L2_5 普战
  -> L3_5 普战
  -> L4_5 道路（绕过迷宫主入口，直达出口）
  -> L5_3 普战
  -> L6_3 普战
  -> L7_3 普战
  -> BOSS
```

**特点**：避开了北线的 Layer 4 迷宫主入口，但南线本身不再是"干净路线"——L2_5 和 L3_5 各有一条岔路通入陷阱。走对的话确实最快，但走错的代价和北线一样。

---

### 路径B：北线绞杀（必踩迷宫）

```
START
  -> L1_1 道路
  -> L2_1 普战
  -> L3_1 艰战
  -> L4_1 普战（进入迷宫）
  -> L4_2 废墟
  -> L4_3 突发（陷阱，发现不通）
  -> L4_4 艰战
  -> L4_5 道路（终于找到出口）
  -> L5_1 突发
  -> L6_1 艰战
  -> L7_1 艰战
  -> BOSS
```

**特点**：被强制送入 Layer 4 迷宫，在 L4_1~L4_4 之间绕路，多消耗 3~4 步体力。Layer 4 的 L4_3 是经典伪前进——从 Layer 3 进来以为在推进，实际进了迷宫死角。

---

### 路径C：贪婪探索线（死路全探）

```
START
  -> L1_3 突发
  -> L1_4 废墟（死路，搜刮后返回）
  -> L1_3
  -> L2_3 黑市
  -> L2_4 废墟（死路，返回）
  -> L2_3
  -> L3_3 委托（接取）
  -> L4_3 突发（陷阱，退回）
  -> L3_3
  -> L2_3
  -> L3_2 安全屋（恢复）
  ...（后续找路出迷宫）
```

**特点**：接了委托，探了 2 条死路，踩了 Layer 4 陷阱，体力消耗巨大。适合二周目熟悉地图后的资源最大化路线。

---

### 路径D：南线第一岔路（L2_5 误入死路）

```
START
  -> L1_5 道路
  -> L2_5 普战
  -> L2_6 突发（**南线岔路1：拐入死路**）
  -> L3_4 道路（死路，无法前进）
  -> L2_6（返回）
  -> L2_5
  -> L3_5 普战
  ...（后续回正路）
```

**特点**：从 L2_5 出发时，眼前有两条路：向前去 L3_5（正路），横向去 L2_6（岔路）。L2_6 看起来连着一个 Layer 3 节点，像是正常推进，但 L3_4 是死路。多浪费 4 步体力。

---

### 路径E：南线第二岔路（L3_5 误入迷宫陷阱）

```
START
  -> L1_5 道路
  -> L2_5 普战
  -> L3_5 普战
  -> L4_3 突发（**南线岔路2：拐入迷宫陷阱**）
  -> L4_2 废墟（发现被困在迷宫里）
  -> L4_1 普战
  -> L3_1 艰战（从迷宫另一端退回 Layer 3）
  ...（重新找路去 L4_5）
```

**特点**：L3_5 有 3 条出边：L4_3(陷阱)、L4_4(艰战)、L4_5(出口)。选择 L4_3 的玩家以为自己抄了近路，实际掉进了 Layer 4 绞杀迷宫，被迫在 L4_1~L4_4 之间绕圈，最后从 L3_1 或 L4_4 才能逃出。这是全图最阴险的设计——**捷径的尽头是迷宫**。

## 设计说明

1. **Layer 4 绞杀**：整个 Layer 4 是一个"四通迷宫"，但只有一个出口 L4_5。玩家无论从 L3_1、L3_2 还是 L3_5 进入，都得在里面绕到 L4_5。这是全图最狠的设计——不是死路，而是**迷宫**。
2. **L4_3 的三重陷阱**：L4_3 现在有三个入口——L3_3（委托线）、L3_5（南线岔路）、L4_2/L4_4（迷宫内部）。无论从哪边进来，都发现自己只连迷宫节点，不连 Layer 5。南线玩家尤其容易中招，因为从 L3_5 看，L4_3 和 L4_5 都是"向前的节点"，选错就直接掉进绞杀迷宫。
3. **Layer 6 的 50% 赌博**：从 Layer 5 进入 Layer 6 的 4 个节点中，L6_1/L6_3 通 Boss，L6_2/L6_4 进死路。没有提示的话，玩家在 Layer 5 选择出口时就像在抽盲盒。
4. **安全屋位置**：L3_2 在 Layer 4 迷宫入口前，L5_4 在 Layer 6 赌博前。两个安全屋都卡在"关键决策点"之前，给玩家一次恢复和重新考虑的机会。
