### 概述

前四章中，每一章都会有一位焦心的委托人丢失了一封信件，希望兰奈希尔能将其寻回。

接取委托之后，兰奈希尔需要在当前的地图中探索，找到“遗失的信件”并将其带回给委托人（委托人会一直处在“委托任务”节点）

任务完成后，“委托任务”节点将转变为“普通道路”节点

注意：一旦选择前往下一章，本章的“委托任务”就将失败，不会获得任何奖励或惩罚。

### 任务流程

接取委托任务后，将在符合要求的随机节点中刷新出“遗失的信件”；

“遗失的信件”需要占据1×1的存储空间；并且需要注意：每进入新的一章，都会自动移除存储空间中的“遗失的信件”。

“遗失的信件”可能在以下节点获得：

|  节点  | 概率  |                   说明                   |
| :--: | :-: | :------------------------------------: |
| 艰难战斗 | 70% |    作为战斗胜利的战利品，出现在任何一个“艰难作战”节点中的概率均等    |
|  黑市  | 15% | 作为商品出售，对应的黑市会有一个额外货槽；出现在任何一个“黑市”中的概率均等 |
| 安全屋  | 10% |     进入安全屋后自动获得；出现在任何一间“安全屋”中的概率均等      |
| 普通战斗 | 5%  |    作为战斗胜利的战利品，出现在任何一个“普通作战”节点中的概率均等    |
黑市中，“遗失的信件”售价如下：

| 章节  | 售价  |
| :-: | :-: |
| 第一章 | 18  |
| 第二章 | 29  |
| 第三章 | 40  |
| 第四章 | 52  |
### 奖励

寻回信件之后，委托人将会拿出颇为丰厚的谢礼赠予兰奈希尔：有防身用的武器、背包、充足的道具、也有可能是足量的财产、珍贵的信物……

奖励中会出现“信物”、“金币”、“武器”、“背包”、“消耗品”五个选项，可以任选两项；

其中：
若选择奖励为“武器”或“背包”：凡是本次冒险中未获得过，且已经解锁的“武器”和“背包”均有可能出现，且出现概率均等；
若选择奖励为“信物”：若是第一章、第二章的任务，则完成后会获得1件信物；完成第三章、第四章的任务会获得2件信物；凡是本次冒险中未获得的信物均有可能出现，且出现的概率均等
若选择奖励为“消耗品”：第一章的任务完成后获得4件、第二章6件、第三章9件、第四章11件，抽取概率如下：

| 消耗品种类 | 概率  |
| :---: | :-: |
|  磨刀石  | 22% |
|  石块   | 11% |
| 能量饮料  | 29% |
|  手电筒  | 15% |
|  火把   | 17% |
| 安全屋房卡 | 6%  |
若选择奖励为“金币”：每章完成委托任务后获得金币数量如下所示：

| 章节  | 可以获得的金币数量 |
| :-: | :-------: |
| 第一章 |    22     |
| 第二章 |    33     |
| 第三章 |    44     |
| 第四章 |    55     |

完成委托任务后，除了上述五选二的奖励之外，还能够从委托者手中获取一封“幸存者的信件”；

无论手中持有几封“幸存者的信件”，都需要占据1×1的存储空间。

### 真结局

集齐4封“幸存者的信件”是进入真结局的必要条件。
