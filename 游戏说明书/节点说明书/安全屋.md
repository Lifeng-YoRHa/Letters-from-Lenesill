安全屋节点可供你休息，补充道具，修理武器；可以反复进入

每次进入安全屋节点都需要一张房卡，无论是否曾进入过这间安全屋

每间安全屋内都一定包含如下设施：

一张床：恢复所有体力

一个铁砧：完全修理武器，恢复所有耐久，并将其攻击力回复至初始值（可使用1次，随幸存者笔记撰写可提升至2次）
（每间安全屋的铁砧使用次数独立）

一个冰箱：内含2瓶能量饮料（随幸存者笔记撰写可提升至3瓶）

一个零钱罐：内含若干金币，随章节而渐增。

此外，还有随机散落的消耗性道具1件，随幸存者笔记撰写可提升至3件。

除了床之外，所有的设施都不会随着再次进入而补充使用次数。

零钱罐内可能出现的金币数量（每个数量出现的概率均为20%）：随幸存者笔记撰写可以提高1枚（如：提升后第一章的零钱罐可能出现金币数量变为7/8/9/10/11）

| :-: | :------------: |

| 第二章 |  8/9/10/11/12  |
| 第三章 | 10/11/12/13/14 |
| 第四章 | 12/13/14/15/16 |
| 最终章 | 14/15/16/17/18 |
