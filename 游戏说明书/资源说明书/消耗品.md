消耗品是协助冒险的一系列有用的工具，分为如下几种：

1.磨刀石（大小：1×1）：恢复武器的3点耐久（随幸存者笔记撰写可提升至5点），并降低武器的1点攻击力（武器攻击力不会低于4！）

2.能量饮料（大小：1×1）：恢复7点体力（随幸存者笔记撰写可提升至10点）

3.石块（大小：1×1）：消耗2点体力丢出，吸引敌人注意力，逃离当前战斗并回到上一个节点

4.手电筒（大小：1×2）：探查随机2个节点的类型（随幸存者笔记撰写可提升至4个）

5.安全屋房卡（大小：1×1）：打开一间“安全屋”

6.火把（大小：1×2）：消耗2点体力丢出，对一名敌人造成30点伤害

消耗品可以在“废墟搜刮”中找到，或者在“黑市交易”与“安全屋”中得到补充，亦可以在战斗结束后作为战利品掉落，或者作为委托任务的奖励；

消耗品都需要放在背包或口袋中，或者也可以拿到的第一时间就使用；

所有消耗品均为一次性物品。

初始可以从3项随机出现的消耗品中选择1项携带，随幸存者笔记撰写可以提高选择次数到3次。

初始时每次选择，出现各种消耗品的概率如下

| 消耗品种类 | 概率  |
| :---: | :-: |
|  磨刀石  | 21% |
|  石块   | 16% |
| 能量饮料  | 32% |
|  手电筒  | 15% |
|  火把   | 12% |
| 安全屋房卡 | 4%  |
